# Prompt Templates

## Speech-Only Base

```text
업로드한 동영상의 음성만 기준으로 일본어 자막 SRT를 작성해 주세요.

규칙:
- 출력은 코드블록 하나의 SRT만.
- 대사, 나레이션 본문, 노래/가사는 포함.
- 화면에 적힌 글자, 크레딧, 텔롭은 제외.
- 효과음, 배경음 설명, 괄호 지시, 화자 라벨은 제외.
- 짧은 감탄사, 독립 발화, 미완문도 들리면 포함.
- 서로 다른 발화는 함부로 합치지 말고 분리할 것.
- 들리는 내용을 임의로 요약하거나 생략하지 말 것.
- 타임스탬프는 실제 발화 흐름에 맞게 SRT 형식으로 작성.
- 불확실한 부분도 가장 가능성 높은 일본어로 적고 빈 설명 문구는 쓰지 말 것.
```
